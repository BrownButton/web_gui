1. Monitoring Parameters 를 확장하면 다른 디바이스들의 박스 크기도 조정된다
2. Monitor 버튼을 클릭하면 카드형 디바이스 들의 사이즈가 툭툭 튀면서 변화된다.
3. All Device Control 좀더 간편하게 수정
4. Modbus 탭 Start Address, Write Value 를 HEX/DEC 값으로 입력할수있도록 수정
5. Parameters의 Value가 이전 값이 남아있음 -> 장비를 연결하지 않았을때도 문제이지만 장비를 연결하고 나서 값을 읽어오기전에 이값이 실제 장비에 적용된 값이라고 헷갈릴 여지가 있음
6. Parameters에서 장비를 선택하면 자동으로 Read All -> 그런데 값을 읽어오던중 장비를 바꾸면 바꾸기 전 장비를 읽어오던 프로세스 취소하고 현재 장비를 읽어오도록 수정
7. Parameters - Parameter List에서 필터를 바꾸면 값을 다시 refresh 하도록 수정
8. Parameters - Parameter List에서 값을 선택해서 Live Watch 기능 추가
9. Live Watch 폴링 주기 설정에 추가
