# 2-4. FC(10) Write Multiple Registers 동작 시험

| 항목 | 내용 |
|------|------|
| **분류** | Modbus RTU |
| **No** | 2-4 |

## 시험 목적

Modbus RTU Write Multiple Registers [0x10] 명령 입력 시 복수의 레지스터가 정상적으로 기록되는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Modbus Serial Test 프로그램과 드라이브를 연결한다.
   - Hercules v3.2.8 테스트 프로그램 활용
   - 프로그램의 Serial 탭 클릭 후 우측 Serial 항목을 설정한다.
     > Name: COM6 (\*PC 환경에 따라 다른 COM 포트가 할당될 수 있음)
     > Baud: 19200bps, Data Size: 8, Parity: even
3. Open 버튼을 클릭하여 COM Port의 Opened 여부를 확인한다.
4. Write Multiple Registers [0x10] 명령으로 연속된 복수 레지스터에 값을 기록한다.

**요청 프레임 예시 (Operating Mode [0xD106]과 Running Direction [0xD102] 2개 연속 쓰기)**

| 필드 | Node ID | Function | Start Addr Hi | Start Addr Lo | Qty Hi | Qty Lo | Byte Count | Data... | CRC |
|------|---------|----------|--------------|--------------|--------|--------|------------|---------|-----|
| 내용 | 0x01 | 0x10 | 0xD1 | 0x02 | 0x00 | 0x02 | 0x04 | 데이터 4바이트 | - |

5. 명령 코드 송신 후, 수신되는 응답 프레임을 확인한다.

**응답 프레임**

| 필드 | Node ID | Function | Start Addr Hi | Start Addr Lo | Qty Hi | Qty Lo | CRC |
|------|---------|----------|--------------|--------------|--------|--------|-----|
| 내용 | 0x01 | 0x10 | 0xD1 | 0x02 | 0x00 | 0x02 | - |

6. FC(03)으로 기록된 레지스터 값들을 읽어 정상 기록되었는지 확인한다.

7. 잘못된 값 송신 시 에러코드를 확인한다.

**에러 응답 프레임**

| 바이트 | 0 | 1 | 2 | 3 | 4 |
|--------|------|------|---------|---|---|
| 내용 | 0x01 | 0x90 | 0x01~0x06 | - | - |

## 판정기준

- 복수 레지스터 쓰기 명령에 대한 응답이 정상으로 나타나면 합격
- FC(03) 읽기를 통해 기록된 값이 요청값과 일치함을 확인

## 시험결과

- [ ] 합격 / 불합격

## 기타

**Hercules 프로그램 사전 설정**
- Serial 탭 이동 → Receive/Sent data 화면 우클릭 → Hex Enable 체크 → 송신 메세지 HEX 체크
