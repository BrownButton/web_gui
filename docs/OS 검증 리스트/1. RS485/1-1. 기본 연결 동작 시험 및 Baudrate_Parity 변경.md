# 1-1. 기본 연결 동작 시험 및 Baudrate / Parity / Stop Bit 변경

| 항목 | 내용 |
|------|------|
| **분류** | RS485 |
| **No** | 1-1 |

## 시험 목적

EC FAN과 RS485 Modbus 통신 연결이 정상적으로 이루어지는지 확인하고,
Baudrate / Parity / Stop Bit 변경 후에도 정상적으로 통신이 이루어지는지 검증한다.
시험 완료 후 초기 통신 설정으로 복원한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
- USB to RS485 Converter
- EC FAN Configurator (EC FAN Control 프로그램)

> ⚠️ **단일 장비 전제**: 이 시험은 장비 1대만 연결된 환경에서 진행한다.
> 통신 설정을 변경하면 동일 버스에 연결된 다른 장비는 응답하지 않게 되므로,
> 멀티 장비 환경에서는 반드시 해당 장비를 버스에서 분리 후 진행한다.

## 통신 확인 방법

각 케이스에서 통신 정상 여부는 아래 파라미터 중 하나 이상을 Modbus 명령으로 읽어 응답이 정상 수신되면 합격으로 판정한다.

**기본 확인 파라미터 (기본값):**

| 파라미터 | FC | 레지스터 |
|----------|----|---------|
| Device Name | FC03 | (기기별 확인) |
| Main SW Ver | FC03 | (기기별 확인) |
| Main Boot Ver | FC03 | (기기별 확인) |
| Inverter SW Ver | FC03 | (기기별 확인) |
| Inverter Boot Ver | FC03 | (기기별 확인) |

> 시험자가 위 중 일부를 선택하거나 다른 레지스터를 지정하여 진행할 수 있다.

---

## 시험 방법

### 지원 통신 설정 (시험 전 확인)

아래 값은 드라이브 사양서 기준이다. 미지원 항목은 해당 케이스를 생략하고 결과표에 "미지원"으로 기록한다.

| 항목 | 지원 값 |
|------|---------|
| Baudrate | 9600 / **19200** / 38400 / 57600 / 115200 (사양서 확인 필요) |
| Parity | **Even** / Odd / None |
| Stop Bit | **1** / 2 |

---

### 기본 접속 확인

1. USB to RS485 컨버터를 이용하여 EC FAN과 연결한다.
2. 프로그램을 초기값(Baudrate: 19200, Parity: Even, Stop Bit: 1)으로 접속한다.
3. 통신 확인 파라미터를 읽어 응답 정상 수신 여부를 확인하고 값을 기록한다.

---

### Baudrate 변경 시험

각 Baudrate에 대해 아래 절차를 반복한다:

```
① 현재 접속 상태에서 Setting 메뉴 → Baudrate를 변경 대상 값으로 설정
② Save to Memory
③ 전원 재투입 (물리적 전원 Off → 3초 대기 → On)
④ 프로그램에서 새 Baudrate로 재접속
⑤ 통신 확인 파라미터 읽기 → 응답 정상 수신 확인
⑥ Baudrate를 19200(기본값)으로 복구 → Save to Memory → 전원 재투입 → 정상 접속 확인
```

대상: **38400 bps** (최소 1개 이상, 여건에 따라 지원 값 전체 진행)

---

### Parity 변경 시험

각 Parity에 대해 아래 절차를 반복한다:

```
① Setting 메뉴 → Parity를 변경 대상 값으로 설정 (Stop Bit 동시 조정)
② Save to Memory
③ 전원 재투입
④ 프로그램에서 새 Parity / Stop Bit로 재접속
⑤ 통신 확인 파라미터 읽기 → 응답 정상 수신 확인
⑥ Parity를 Even(기본값), Stop Bit를 1로 복구 → Save → 전원 재투입 → 정상 접속 확인
```

| 단계 | Parity | Stop Bit |
|------|--------|----------|
| P-1 | Odd | 1 |
| P-2 | None | 2 |

---

### 최종 복원 확인

모든 시험 완료 후:

4. Setting 메뉴에서 통신 설정이 초기값(Baudrate: 19200, Parity: Even, Stop Bit: 1)인지 확인한다.
5. 초기값과 다를 경우 초기값으로 설정 후 Save to Memory → 전원 재투입한다.
6. 초기값으로 재접속하여 통신 확인 파라미터 정상 수신을 확인한다.

---

## 판정기준

- 각 통신 설정으로 변경 후 재접속 시 Modbus 응답 정상 수신
- 시험 종료 후 초기값(19200 / Even / Stop Bit 1)으로 정상 복원
