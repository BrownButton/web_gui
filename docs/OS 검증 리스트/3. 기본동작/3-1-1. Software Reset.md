# 3-1-1. Software Reset

| 항목 | 내용 |
|------|------|
| **분류** | 기본동작 |
| **No** | 3-1-1 |

## 시험 목적

Modbus 명령을 통해 드라이브 소프트웨어 리셋(Software Reset)이 정상 동작하는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. Open 버튼을 클릭하여 COM Port 연결을 확인한다.
4. 드라이브의 정상 동작 상태를 확인한다. (파라미터 읽기 가능 여부 확인)
5. Software Reset 명령을 아래 프레임을 참조하여 입력한다.

**SW Reset 명령 (0xD000 레지스터에 Reset 코드 쓰기)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD0 | 0x00 | Reset Code Hi | Reset Code Lo | CRC Hi | CRC Lo |

6. 리셋 명령 전송 후 드라이브가 재시작되는지 확인한다.
   - 통신 응답이 일시적으로 끊겼다가 재연결되는 현상 확인
7. 재시작 후 파라미터를 읽어 정상 동작 상태인지 확인한다.

## 판정기준

- SW Reset 명령 전송 후 드라이브가 정상적으로 재시작됨
- 재시작 후 통신 및 파라미터 읽기가 정상 동작

## 시험결과

- [ ] 합격 / 불합격

## 기타

**Hercules 프로그램 사전 설정**
- Serial 탭 이동 → Receive/Sent data 화면 우클릭 → Hex Enable 체크 → 송신 메세지 HEX 체크

SW Reset 전/후 통신 상태 데이터 캡처 첨부
