# 3-1-2. Alarm Reset

| 항목 | 내용 |
|------|------|
| **분류** | 기본동작 |
| **No** | 3-1-2 |

## 시험 목적

알람(Fault) 상태에서 Reset 명령 전송 시 알람이 해제되고 정상 운전 가능 상태로 복귀하는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. Open 버튼을 클릭하여 COM Port 연결을 확인한다.
4. 알람 상태를 유발한다.
   - 예시) 모터 구속 상태에서 구동 명령 전송, 또는 보호동작 시험 중 발생한 알람 활용
5. Motor Status [0xD011]을 읽어 에러 비트(Bit 4: FB)가 Set 되어 있는지 확인한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

6. Alarm Reset 명령을 전송한다.

**Alarm Reset 명령 (0xD000 레지스터에 Reset 코드 쓰기)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD0 | 0x00 | Alarm Reset Code Hi | Alarm Reset Code Lo | CRC Hi | CRC Lo |

7. Alarm Reset 후 Motor Status [0xD011]을 다시 읽어 에러 비트가 Clear 되었는지 확인한다.
8. 정상 운전 명령을 전송하여 구동 가능 여부를 확인한다.

## 판정기준

- Alarm Reset 명령 전송 후 Motor Status의 에러 비트(FB, 기타 에러 비트)가 Clear됨
- 알람 해제 후 정상 운전 가능 상태로 복귀

## 시험결과

- [ ] 합격 / 불합격

## 기타

알람 유발 방법 및 Reset 전/후 Motor Status 데이터 캡처 첨부
