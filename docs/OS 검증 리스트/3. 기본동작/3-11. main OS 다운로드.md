# 3-10-1. Main OS 다운로드

| 항목 | 내용 |
|------|------|
| **분류** | 기본동작 |
| **No** | 3-10-1 |

## 시험 목적

Main OS 다운로드 완료 후 전원을 Off/On하면 Flash 메모리에 있는 새로운 OS가 정상 적용되는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- EC FAN OS 업데이트 툴

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 서로 연결한다.
2. EC FAN OS 업데이트 툴을 실행한다.
3. 통신 사양을 설정한 후 Connect 버튼을 누른다.
   - (Baudrate 19,200bps, Even Parity 조건은 공장 출하 상태에서의 초기 값이며, 이 값이 팬 드라이브 설정과 다를 경우 접속이 되지 않으므로 유의)
4. 다운로드할 **Main** F/W 파일을 선택한 후 Forced Download 체크박스를 체크한다.
5. Start 버튼을 눌러 다운로드를 진행한다.
   - 다운로드 진행 중 진행 상황을 모니터링한다.
6. 다운로드 완료 후 전원을 Off/On한다.
7. 통신을 재접속하여 Main SW 버전을 확인한다.
   - LED 점멸 주기 변화를 통해서도 업데이트 확인 가능

## 판정기준

- Main OS 다운로드 및 전원 Off/On 후 펌웨어 버전이 업데이트된 버전으로 확인됨
- (Main OS의 경우 LED 점멸 주기 변화를 통해 업데이트 확인)

## 시험결과

|------|-----------|-----------|------|
| Main SW Ver (업데이트 전) | - | - | - |
| Main SW Ver (업데이트 후) | - | - | 합격 / 불합격 |

## 기타

- Main 및 Inverter 모두 F/W 업데이트 시험을 진행한다.

A) EC FAN OS 업데이트 방법 (스크린샷 첨부)

B) MAIN부 OS 업데이트 결과 (점멸 주기 변화 확인 사진 첨부)
