# 5-16. Error 0x42 RST Power Fail (주 전원 결상 보호)

| 항목 | 내용 |
|------|------|
| **분류** | 보호동작 |
| **No** | 5-16 |

## 시험 목적

주 전원(RST)이 결상 되었을 때 알람 동작 여부를 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Modbus Serial Test 프로그램과 드라이브를 연결한다.
   - Hercules v3.2.8 테스트 프로그램 활용
   - Baud: 19200bps, Data Size: 8, Parity: even
3. Open 버튼을 클릭하여 COM Port 연결을 확인한다.
4. 아래 표와 같이 주 전원을 결상시킨다.

| Test Case | R | S | T |
|-----------|---|---|---|
| Case 1 | O | O | X |
| Case 2 | O | X | O |
| Case 3 | X | O | O |
| Case 4 | O | X | X |
| Case 5 | X | X | O |
| Case 6 | X | O | X |
| Case 7 | X | X | X |

5. 전원을 켠 후 아래의 표를 참조하여, Read Input Register [0x04] 명령어를 입력한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

→ `01 04 D0 11 00 01 59 0F` 입력

6. 응답 코드를 읽어 **0번 비트(PHA)**와 **4번 비트(FB)** 가 Set 되어 있는지 확인한다.

## 판정기준

- [0xD011] Motor Status의 값 중 **0번 비트(PHA)와 4번 비트(FB)** 가 Set 되어 있을 경우 합격
- 예상 Motor Status 값: 0x0011

## 시험결과

| Test Case | Motor Status 값 | 판정 |
|-----------|----------------|------|
| Case 1 (T 결상) | - | 합격 / 불합격 |
| Case 2 (S 결상) | - | 합격 / 불합격 |
| Case 3 (R 결상) | - | 합격 / 불합격 |
| Case 4 (S, T 결상) | - | 합격 / 불합격 |

## 기타

**Motor Status 파라미터 비트맵**

| 비트 | 이름 | 설명 |
|------|------|------|
| 0 | PHA | 주 전원 결상 |
| 4 | FB | 팬 에러 (에러 상태) |

A) Motor Status 파라미터 비트맵

B) 송/수신 데이터 (0x0011) 캡처 첨부
