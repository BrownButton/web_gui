# 5-27. Warning 0x40 Torque Limit

| 항목 | 내용 |
|------|------|
| **분류** | 보호동작 (경고) |
| **No** | 5-27 |

## 시험 목적

드라이브 출력 토크가 토크 제한값에 도달했을 때 Warning 신호가 발생하는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. Max Coil Current [0xD13B] 파라미터를 낮은 값으로 설정하여 토크 제한을 낮춘다.
4. Closed-loop Velocity 모드로 구동하면서 부하를 증가시킨다.
   - 또는 팬 블레이드를 부착한 상태에서 높은 속도로 구동
5. Command Torque [0xD050] 파라미터를 모니터링하여 토크 제한값에 도달하는지 확인한다.
6. Warning Status 파라미터에서 비트 6 (Torque Limit Warning, 0x40)이 Set 되었는지 확인한다.

**Warning Status 파라미터 읽기**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | Warning Addr Hi | Warning Addr Lo | 0x00 | 0x01 | CRC Hi | CRC Lo |

## 판정기준

- 출력 토크가 토크 제한값에 도달했을 때 Warning 비트(0x40) Set 확인

## 시험결과

- [ ] 합격 / 불합격

## 기타

토크 제한 도달 시 Command Torque 및 Warning Status 파라미터 데이터 첨부
