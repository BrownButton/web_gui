# 5-14. Error 0x40 Under Voltage (저 전압 에러)

| 항목 | 내용 |
|------|------|
| **분류** | 보호동작 |
| **No** | 5-14 |

## 시험 목적

DC Link 전압이 한계값 이하로 내려갔을 경우 알람 동작 여부 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)
- 슬라이닥스 (전압 가변용)

## 시험 방법

1. 전압을 가변하기 위해 슬라이닥스를 이용하여 전원(380Vac)을 인가한다.
2. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
3. Modbus Serial Test 프로그램과 드라이브를 연결한다.
   - Hercules v3.2.8 테스트 프로그램 활용
   - Baud: 19200bps, Data Size: 8, Parity: even
4. Open 버튼을 클릭하여 COM Port 연결을 확인한다.
5. 아래의 표를 참조하여, Read Input Register [0x04] 명령어를 입력한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

→ `01 04 D0 11 00 01 59 0F` 입력

6. 0xD011의 값이 0임을 확인한다 (정상 상태).
7. 슬라이닥스를 이용하여 220Vac까지 전압을 낮춘다.
8. Motor Status [0xD011]을 다시 읽어 에러 비트를 확인한다.
9. 응답 코드를 읽어 4번 비트(FB)와 12번 비트(UzLow)가 Set 되어 있는지 확인한다.

## 판정기준

- [0xD011] Motor Status의 값 중 **4번 비트(FB)와 12번 비트(UzLow)** 가 Set 되어 있을 경우 합격
- 예상 Motor Status 값: 0x1010

## 시험결과

- [ ] 합격 / 불합격

## 기타

**Motor Status 파라미터 비트맵**

| MSB | ... | UzLow (12) | ... | RL_Cal (10) | ... | N_Limit (8) | BLK (7) | HLL (6) | FB (4) | PHA (0) | LSB |
|-----|-----|-----------|-----|------------|-----|------------|---------|---------|--------|---------|-----|

- UzLow: DC링크 전압 낮음
- FB: 팬 에러 (현재 에러 상태일 경우 활성화)

송/수신 데이터 캡처 (예: 0x1010) 첨부
