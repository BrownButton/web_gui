# 5-15. Error 0x41 Over Voltage (과 전압 에러)

| 항목 | 내용 |
|------|------|
| **분류** | 보호동작 |
| **No** | 5-15 |

## 시험 목적

DC Link 전압이 과전압 임계값을 초과했을 때 드라이브가 보호 동작(알람)을 수행하는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)
- 슬라이닥스 (전압 가변용)

## 시험 방법

1. 슬라이닥스를 이용하여 전원(380Vac 이상)을 인가한다.
2. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
3. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
4. Open 버튼을 클릭하여 COM Port 연결을 확인한다.
5. 정상 전압(380Vac)에서 Motor Status [0xD011] 값이 0임을 확인한다.
6. 슬라이닥스를 이용하여 과전압 임계값 이상으로 전압을 높인다.
7. Motor Status [0xD011]을 읽어 에러 비트 Set 여부를 확인한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

8. Error 코드 파라미터를 읽어 0x41 (Over Voltage)가 발생했는지 확인한다.
9. 전압 정상화 후 Alarm Reset을 실행하여 정상 복귀 확인한다.

## 판정기준

- 과전압 조건 발생 시 드라이브가 보호 동작을 수행함
- Error 코드 0x41 (Over Voltage) 확인

## 시험결과

- [ ] 합격 / 불합격

## 기타

- 과전압 임계값 확인 후 적절한 시험 전압 설정
- 에러 발생 시 Motor Status 및 DC Link 전압 파라미터 데이터 첨부
