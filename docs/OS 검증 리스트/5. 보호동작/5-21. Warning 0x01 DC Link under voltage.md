# 5-21. Warning 0x01 DC Link Under Voltage

| 항목 | 내용 |
|------|------|
| **분류** | 보호동작 (경고) |
| **No** | 5-21 |

## 시험 목적

DC Link 전압이 경고 임계값 이하로 내려갔을 때 Warning 신호가 발생하는지 확인한다.
(에러(알람)와 달리 Warning은 드라이브 운전을 즉시 중단하지 않고 경고 상태를 알림)

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)
- 슬라이닥스 (전압 가변용)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. 정격 전압(380Vac)에서 정상 상태 확인 후, 슬라이닥스로 전압을 서서히 낮춘다.
4. Warning 임계값에 해당하는 전압 구간에서 Warning 파라미터를 읽는다.
5. Warning Status 파라미터에서 비트 0 (DC Link Under Voltage Warning)이 Set 되었는지 확인한다.

**Warning Status 파라미터 읽기**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | Warning Addr Hi | Warning Addr Lo | 0x00 | 0x01 | CRC Hi | CRC Lo |

6. Warning 상태에서도 드라이브 운전이 계속되는지(또는 제한되는지) 확인한다.

## 판정기준

- DC Link 전압이 Warning 임계값 이하로 내려갔을 때 Warning 비트(0x01) Set 확인
- Warning 상태에서의 드라이브 동작 정책 확인 (운전 계속 또는 속도 제한)

## 시험결과

- [ ] 합격 / 불합격

## 기타

Warning 임계 전압 구간에서의 Warning Status 파라미터 데이터 첨부
