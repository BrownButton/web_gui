# 4-1-2. Set Value Source - Analog V (전압)

| 항목 | 내용 |
|------|------|
| **분류** | 구동동작 |
| **No** | 4-1-2 |

## 시험 목적

Set Value Source를 Analog 전압 입력으로 설정 시 아날로그 전압 신호에 의해 팬 모터가 정상 구동되는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
- USB to RS485 Converter
- 가변 전압 소스 (0~10V 출력)
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. Set Value Source를 Analog V 입력으로 설정한다.

**Set Value Source 파라미터 쓰기 (Analog V 모드)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | SV Src Addr Hi | SV Src Addr Lo | 0x00 | Analog V Code | CRC Hi | CRC Lo |

4. 가변 전압 소스를 EC FAN Analog 입력 단자에 연결한다.
5. 다양한 전압값(예: 1V, 5V, 9V)을 인가한다.
6. 각 전압에 따라 팬 모터가 해당 속도 또는 토크로 구동되는지 확인한다.
7. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

## 판정기준

- 아날로그 입력 전압에 비례하여 팬 모터 속도/토크가 정상 제어됨

## 시험결과

| 입력 전압 | 기대 출력 | 실측 출력 | 판정 |
|-----------|-----------|-----------|------|
| 1V | - | - | 합격 / 불합격 |
| 5V | - | - | 합격 / 불합격 |
| 9V | - | - | 합격 / 불합격 |

## 기타

아날로그 입력 전압 및 실제 속도 파라미터 데이터 첨부
