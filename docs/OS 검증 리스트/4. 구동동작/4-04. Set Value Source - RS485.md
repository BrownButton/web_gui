# 4-1-4. Set Value Source - RS485

| 항목 | 내용 |
|------|------|
| **분류** | 구동동작 |
| **No** | 4-1-4 |

## 시험 목적

Set Value Source를 RS485 통신으로 설정 시 Modbus 명령에 의해 팬 모터 속도/토크가 정상 제어되는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. Set Value Source를 RS485 입력으로 설정한다.

**Set Value Source 파라미터 쓰기 (RS485 모드)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | SV Src Addr Hi | SV Src Addr Lo | 0x00 | RS485 Code | CRC Hi | CRC Lo |

4. Set Point [0xD001]을 통해 다양한 속도 명령을 전송한다.
5. 각 Set Point 값에 따라 팬 모터가 해당 속도/토크로 구동되는지 확인한다.
6. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

**Actual Speed 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x2D | 0x00 | 0x01 | CRC Hi | CRC Lo |

## 판정기준

- RS485 Set Point 명령에 비례하여 팬 모터 속도/토크가 정상 제어됨
- Actual Speed 파라미터가 명령값에 정상 추종

## 시험결과

| Set Point | 기대 속도(RPM) | 실측 속도(RPM) | 판정 |
|-----------|---------------|---------------|------|
| - | - | - | 합격 / 불합격 |
| - | - | - | 합격 / 불합격 |
| - | - | - | 합격 / 불합격 |

## 기타

Set Point 및 Actual Speed 파라미터 데이터 캡처 첨부
