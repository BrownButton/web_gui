# 4-3. 구동 모드 설정 (Operation Mode: Torque / Velocity)

| 항목 | 내용 |
|------|------|
| **분류** | 구동동작 |
| **No** | 4-3 |

## 시험 목적

Operating Mode 파라미터 설정에 따라 토크 제어(Open-loop) 및 속도 제어(Closed-loop Velocity) 모드가 정상 동작하는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

### Open-loop (Torque) 모드 전환

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
3. Operating Mode를 Open-loop (0x0002)로 설정한다.

**Operating Mode 쓰기 (Open-loop = 0x0002)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD1 | 0x06 | 0x00 | 0x02 | 0xD1 | 0x36 |

4. Set Point [0xD001]에 10% 토크 명령을 입력한다. (6554 = 0x199A)
5. Command Torque [0xD050] 파라미터를 읽어 10%로 설정되었는지 확인한다.

### Closed-loop Velocity 모드 전환

6. Operating Mode를 Closed-loop Velocity (0x0000)로 설정한다.

**Operating Mode 쓰기 (Closed-loop Velocity = 0x0000)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD1 | 0x06 | 0x00 | 0x00 | 0x50 | 0xF7 |

7. Set Point [0xD001]에 100rpm 속도 명령을 입력한다. (3200 = 0x0C80)
8. Command Velocity [0xD051] 파라미터를 읽어 100rpm으로 설정되었는지 확인한다.
9. Actual Speed [0xD02D]를 읽어 실제 속도가 100rpm에 추종하는지 확인한다.

## 판정기준

- Operating Mode = 0x0002: Open-loop 토크 제어 정상 동작 (Command Torque 추종 확인)
- Operating Mode = 0x0000: Closed-loop 속도 제어 정상 동작 (Actual Speed 추종 확인)

## 시험결과

| 모드 | 설정 | 판정 |
|------|------|------|
| Open-loop (Torque) | Command Torque 10% 확인 | 합격 / 불합격 |
| Closed-loop Velocity | Actual Speed 100rpm 추종 확인 | 합격 / 불합격 |

## 기타

**Hercules 프로그램 사전 설정**
- Serial 탭 이동 → Receive/Sent data 화면 우클릭 → Hex Enable 체크 → 송신 메세지 HEX 체크
