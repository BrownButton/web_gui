# 3-12. inverter OS 다운로드

## 시험 항목

Inverter OS 다운로드 완료 후 전원 Off/On 시 새로운 OS가 정상 적용되는지 확인하는 시험이다.

## 시험 목적

Inverter OS 다운로드 완료 후 전원을 Off/On하면 Flash 메모리에 있는 새로운 OS가 정상 적용되는지 확인한다.

## 시험 방법

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. EC FAN OS 업데이트 툴을 실행한다.
4. 통신 사양을 설정한 후 Connect 버튼을 누른다.
   - (Baudrate 19,200bps, Even Parity 조건은 공장 출하 상태에서의 초기 값이며, 이 값이 팬 드라이브 설정과 다를 경우 접속이 되지 않으므로 유의)
5. 다운로드할 **Inverter** F/W 파일을 선택한 후 Forced Download 체크박스를 체크한다.
6. Start 버튼을 눌러 다운로드를 진행한다.
   - 다운로드 진행 중 진행 상황을 모니터링한다.
7. 다운로드 완료 후 전원을 Off/On한다.
8. 통신을 재접속하여 Inverter SW 버전을 확인한다.

## 판정기준

- Inverter OS 다운로드 및 전원 Off/On 후 Inverter 펌웨어 버전이 업데이트된 버전으로 확인됨
