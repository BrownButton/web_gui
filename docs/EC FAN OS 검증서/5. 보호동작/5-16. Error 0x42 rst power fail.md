# 5-16. Error 0x42 rst power fail

## 시험 항목

주 전원(RST)이 결상 되었을 때 알람 동작 여부를 확인하는 시험이다.

## 시험 목적

주 전원(RST)이 결상 되었을 때 알람 동작 여부를 확인한다.

## 시험 방법

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. 아래 표와 같이 주 전원을 결상시킨다.

| Test Case | R | S | T |
|-----------|---|---|---|
| Case 1 | O | O | X |
| Case 2 | O | X | O |
| Case 3 | X | O | O |
| Case 4 | O | X | X |
| Case 5 | X | X | O |
| Case 6 | X | O | X |
| Case 7 | X | X | X |

4. 전원을 켠 후 아래의 표를 참조하여, Read Input Register [0x04] 명령어를 입력한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

→ `01 04 D0 11 00 01 59 0F` 입력

5. 응답 코드를 읽어 **0번 비트(PHA)**와 **4번 비트(FB)** 가 Set 되어 있는지 확인한다.

**Motor Status 파라미터 비트맵**

| 비트 | 이름 | 설명 |
|------|------|------|
| 0 | PHA | 주 전원 결상 |
| 4 | FB | 팬 에러 (에러 상태) |

## 판정기준

- [0xD011] Motor Status의 값 중 **0번 비트(PHA)와 4번 비트(FB)** 가 Set 되어 있을 경우 합격
- 예상 Motor Status 값: 0x0011
