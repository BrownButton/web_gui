# 5-19. Error 0x53 excessive speed deviation

## 시험 항목

Closed-loop 속도 제어 중 목표 속도와 실제 속도의 편차가 허용 범위를 초과했을 때 보호 동작(알람)이 수행되는지 확인하는 시험이다.

## 시험 목적

Closed-loop 속도 제어 중 목표 속도와 실제 속도의 편차가 허용 범위를 초과했을 때 보호 동작(알람)이 수행되는지 확인한다.

## 시험 방법

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Operating Mode를 Closed-loop Velocity로 설정한다.
4. 목표 속도를 설정한다.
5. 팬을 물리적으로 구속하거나 부하를 가하여 실제 속도가 목표 속도를 따라가지 못하는 조건을 만든다.
6. Motor Status [0xD011]을 읽어 에러 비트 Set 여부를 확인한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

7. Error 코드 파라미터를 읽어 0x53 (Excessive Speed Deviation)이 발생했는지 확인한다.
8. 구속 해제 후 Alarm Reset을 실행하여 정상 복귀 확인한다.

## 판정기준

- 속도 편차 과대 조건 발생 시 드라이브가 보호 동작을 수행함
- Error 코드 0x53 (Excessive Speed Deviation) 확인
