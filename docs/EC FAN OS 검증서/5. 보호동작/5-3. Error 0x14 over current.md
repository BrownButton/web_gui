# 5-3. Error 0x14 over current

## 시험 항목

드라이브 출력 전류가 과전류 임계값을 초과했을 때 보호 동작(알람)이 수행되고 Error 코드 0x14가 출력되는지 확인하는 시험이다.

## 시험 목적

드라이브 출력 전류가 과전류 임계값을 초과했을 때 보호 동작(알람)이 수행되는지 확인한다.

## 시험 방법

> 📌 프로시저(Procedure)에 할당해서 처리하는 방식으로 구현 예정

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Over Current 에러를 유발하는 프로시저(FC 0x2B)를 실행한다.
4. Motor Status [0xD011]을 읽어 에러 비트(Bit 4: FB) Set 여부를 확인한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

5. Error 코드 파라미터를 읽어 0x14 (Over Current)가 발생했는지 확인한다.
6. Alarm Reset 후 정상 동작으로 복귀하는지 확인한다.

## 판정기준

- Over Current 조건 발생 시 드라이브가 보호 동작을 수행함
- Error 코드 0x14 (Over Current) 확인
