# 4-03. Set Value Source - Analog I

## 시험 항목

Set Value Source를 Analog 전류 입력으로 설정 시 아날로그 전류 신호에 의해 팬 모터가 정상 구동되는지 확인하는 시험이다.

## 시험 목적

Set Value Source를 Analog 전류 입력으로 설정 시 아날로그 전류 신호에 의해 팬 모터가 정상 구동되는지 확인한다.

## 시험 방법

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Set Value Source를 Analog I 입력으로 설정한다.

**Set Value Source 파라미터 쓰기 (Analog I 모드)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | SV Src Addr Hi | SV Src Addr Lo | 0x00 | Analog I Code | CRC Hi | CRC Lo |

4. 가변 전류 소스를 EC FAN Analog 입력 단자에 연결한다.
5. 다양한 전류값(예: 4mA, 12mA, 20mA)을 인가한다.
6. 각 전류값에 따라 팬 모터가 해당 속도 또는 토크로 구동되는지 확인한다.
7. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

## 판정기준

- 아날로그 입력 전류에 비례하여 팬 모터 속도/토크가 정상 제어됨
- 4mA: 최소 출력(0%), 20mA: 최대 출력(100%)에 해당
