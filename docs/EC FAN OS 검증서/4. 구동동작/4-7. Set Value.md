# 4-07. Set Value

## 시험 항목

Set Point [0xD001] 파라미터를 통해 다양한 지령값을 설정하고 드라이브가 해당 값에 정상 추종하는지 확인하는 시험이다.

## 시험 목적

Set Point [0xD001] 파라미터를 통해 다양한 지령값을 설정하고 드라이브가 해당 값에 정상 추종하는지 확인한다.

## 시험 방법

### Open-loop 모드 Set Value 검증

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Operating Mode를 Open-loop (0x0002)로 설정한다.
4. 다양한 Set Point 값을 설정하고 각 값에 대한 토크 출력을 확인한다.

| Set Point 값 | 의미 | 기대 토크 |
|-------------|------|-----------|
| 0 | 0% | 0% |
| 6554 (0x199A) | 10% | 10% |
| 32768 (0x8000) | 50% | 50% |
| 65535 (0xFFFF) | 100% | 100% |

- 변환 공식: raw = pct / 100 × 65535

5. Command Torque [0xD050] 파라미터를 읽어 설정값과 일치하는지 확인한다.

### Closed-loop Velocity 모드 Set Value 검증

6. Operating Mode를 Closed-loop Velocity (0x0000)로 설정한다.
7. 다양한 Set Point 값을 설정하고 각 값에 대한 속도 출력을 확인한다.

| Set Point 값 | 기대 속도(RPM) |
|-------------|--------------|
| 0 | 0 RPM |
| 3200 (0x0C80) | 100 RPM |
| 6400 (0x1900) | 200 RPM |
| 32000 (0x7D00) | 1000 RPM |

- 변환 공식: raw = rpm / maxSpeed × 64000 (maxSpeed = 1600 RPM 기준)

8. Actual Speed [0xD02D]를 읽어 설정 속도에 추종하는지 확인한다.

## 판정기준

- Set Point 값에 따라 토크/속도가 정상 비례하여 제어됨
- Command Torque / Actual Speed 파라미터가 기대값에 일치 또는 추종
