# 4-01. Set Value Source - PWM 입력

## 시험 항목

Set Value Source를 PWM 입력으로 설정 시 PWM 신호에 의해 팬 모터가 정상 구동되는지 확인하는 시험이다.

## 시험 목적

Set Value Source를 PWM 입력으로 설정 시 PWM 신호에 의해 팬 모터가 정상 구동되는지 확인한다.

## 시험 방법

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Set Value Source를 PWM 입력으로 설정한다.

**Set Value Source 파라미터 쓰기 (PWM 모드)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | SV Src Addr Hi | SV Src Addr Lo | 0x00 | PWM Code | CRC Hi | CRC Lo |

4. PWM 신호 발생기를 EC FAN PWM 입력 단자에 연결한다.
5. 다양한 듀티비(예: 10%, 50%, 90%)의 PWM 신호를 인가한다.
6. 각 듀티비에 따라 팬 모터가 해당 속도 또는 토크로 구동되는지 확인한다.
7. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

## 판정기준

- PWM 듀티비에 비례하여 팬 모터 속도/토크가 정상 제어됨
