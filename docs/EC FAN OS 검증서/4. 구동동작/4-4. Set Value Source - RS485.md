# 4-04. Set Value Source - RS485

## 시험 항목

Set Value Source를 RS485 통신으로 설정 시 Modbus 명령에 의해 팬 모터 속도/토크가 정상 제어되는지 확인하는 시험이다.

## 시험 목적

Set Value Source를 RS485 통신으로 설정 시 Modbus 명령에 의해 팬 모터 속도/토크가 정상 제어되는지 확인한다.

## 시험 방법

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Set Value Source를 RS485 입력으로 설정한다.

**Set Value Source 파라미터 쓰기 (RS485 모드)**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | SV Src Addr Hi | SV Src Addr Lo | 0x00 | RS485 Code | CRC Hi | CRC Lo |

4. Set Point [0xD001]을 통해 다양한 속도 명령을 전송한다.
5. 각 Set Point 값에 따라 팬 모터가 해당 속도/토크로 구동되는지 확인한다.
6. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

**Actual Speed 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x2D | 0x00 | 0x01 | CRC Hi | CRC Lo |

## 판정기준

- RS485 Set Point 명령에 비례하여 팬 모터 속도/토크가 정상 제어됨
- Actual Speed 파라미터가 명령값에 정상 추종
