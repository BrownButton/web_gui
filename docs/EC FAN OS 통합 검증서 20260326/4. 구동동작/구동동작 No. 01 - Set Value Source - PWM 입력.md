# 구동동작 No. 01 - Set Value Source - PWM 입력

## 시험 항목

Set Value Source를 PWM 입력으로 설정 시 PWM 신호에 의해 팬 모터가 정상 구동되는지 확인하는 시험이다.

## 시험 목표

Set Value Source [0xD101]을 PWM 모드(0x0003)로 설정한 후, PWM 신호 발생기에서 다양한 듀티비를 인가하여 팬 모터 속도/토크가 듀티비에 비례하여 정상 제어되는지 확인한다.

## 시험 기준

### 사전 조건

- RS485 인터페이스를 통해 드라이브와 호스트 PC 연결 완료
- PWM 신호 발생기 준비 완료 (EC FAN PWM 입력 단자에 연결)
- Slave ID: 0x01 기준

### 시험 절차

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Set Value Source를 PWM 입력(0x0003)으로 설정한다.

**Set Value Source 쓰기 (PWM = 0x0003) — FC(06) Write Single Register**
- 레지스터: Set Value Source [0xD101]

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD1 | 0x01 | 0x00 | 0x03 | CRC Lo | CRC Hi |

4. PWM 신호 발생기를 EC FAN PWM 입력 단자에 연결한다.
5. 다양한 듀티비의 PWM 신호를 순서대로 인가한다.

| PWM 듀티비 | 기대 출력 |
|-----------|----------|
| 10% | 최소 구동 수준 |
| 50% | 중간 속도/토크 |
| 90% | 최대 구동 수준 |

6. 각 듀티비에 따라 팬 모터가 해당 속도 또는 토크로 구동되는지 확인한다.
7. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

**Actual Speed 읽기 명령 — FC(04) Read Input Register**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x2D | 0x00 | 0x01 | CRC Lo | CRC Hi |

→ `01 04 D0 2D 00 01 99 CB`

### 판정 기준

- PWM 듀티비에 비례하여 팬 모터 속도/토크가 정상 제어됨
- Actual Speed 파라미터가 PWM 입력에 정상 추종
