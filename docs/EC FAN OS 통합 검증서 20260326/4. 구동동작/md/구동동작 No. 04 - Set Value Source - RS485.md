# 구동동작 No. 04 - Set Value Source - RS485

## 시험 항목

Set Value Source - RS485

## 시험 목표

- Set Value Source [0xD101]을 RS485 모드(0x0001)로 설정한 후 Setpoint [0xD001]에 다양한 지령값을 전송한다.
- RS485 Set Point 명령에 비례하여 팬 모터 속도/토크가 정상 제어됨을 확인한다.
- Actual Speed [0xD02D] 파라미터가 명령값에 정상 추종함을 확인한다.

## 시험 기준

### 사전 조건

- RS485 인터페이스를 통해 드라이브와 호스트 PC 연결 완료
- Slave ID: 0x01 기준

### 시험 절차

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Set Value Source를 RS485(0x0001)로 설정한다.

**Set Value Source 쓰기 (RS485 = 0x0001) — FC(06) Write Single Register**
- 레지스터: Set Value Source [0xD101]

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD1 | 0x01 | 0x00 | 0x01 | CRC Lo | CRC Hi |

4. Set Point [0xD001]을 통해 다양한 속도 명령을 전송한다.

**Set Point 쓰기 (예: 100rpm = 0x0C80) — FC(06) Write Single Register**
- 레지스터: Setpoint [0xD001]
- Speed Control 모드 변환 공식: raw = rpm / maxSpeed × 64000 (maxSpeed = 1600RPM 기준)

| Set Point 값 | 기대 속도 |
|-------------|----------|
| 0x0000 (0) | 0 RPM (정지) |
| 0x0C80 (3200) | 100 RPM |
| 0x3200 (12800) | 400 RPM |
| 0x7D00 (32000) | 1000 RPM |

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD0 | 0x01 | Set Hi | Set Lo | CRC Lo | CRC Hi |

5. 각 Set Point 값에 따라 팬 모터가 해당 속도/토크로 구동되는지 확인한다.
6. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

**Actual Speed 읽기 명령 — FC(04) Read Input Register**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x2D | 0x00 | 0x01 | CRC Lo | CRC Hi |

→ `01 04 D0 2D 00 01 99 CB`

### 판정 기준

- RS485 Set Point 명령에 비례하여 팬 모터 속도/토크가 정상 제어됨
- Actual Speed 파라미터가 명령값에 정상 추종
