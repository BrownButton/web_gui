# 구동동작 No. 03 - Set Value Source - Analog I

## 시험 항목

Set Value Source를 Analog 전류 입력으로 설정 시 아날로그 전류 신호에 의해 팬 모터가 정상 구동되는지 확인하는 시험이다.

## 시험 목표

Set Value Source [0xD101]을 AIN2(Analog I, 0x0002)로 설정한 후, 가변 전류 소스(4~20mA)를 인가하여 팬 모터 속도/토크가 전류에 비례하여 정상 제어되는지 확인한다.

## 시험 기준

### 사전 조건

- RS485 인터페이스를 통해 드라이브와 호스트 PC 연결 완료
- 가변 전류 소스 준비 완료 (4~20mA 범위, EC FAN Analog 입력 단자에 연결)
- Slave ID: 0x01 기준

### 시험 절차

1. RS485 인터페이스를 통해 드라이브와 호스트 PC를 연결한다.
2. 통신 툴을 실행하고 드라이브와 정상 통신되는지 확인한다.
3. Set Value Source를 Analog I 입력(AIN2, 0x0002)으로 설정한다.

**Set Value Source 쓰기 (AIN2 = 0x0002) — FC(06) Write Single Register**
- 레지스터: Set Value Source [0xD101]

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x06 | 0xD1 | 0x01 | 0x00 | 0x02 | CRC Lo | CRC Hi |

4. 가변 전류 소스를 EC FAN Analog 입력 단자에 연결한다.
5. 다양한 전류값을 순서대로 인가한다.

| 입력 전류 | 기대 출력 |
|----------|----------|
| 4mA | 0% (최소 출력 / 정지) |
| 8mA | 25% 출력 |
| 12mA | 50% 출력 |
| 16mA | 75% 출력 |
| 20mA | 100% (최대 출력) |

6. 각 전류값에 따라 팬 모터가 해당 속도 또는 토크로 구동되는지 확인한다.
7. Actual Speed [0xD02D] 파라미터를 읽어 속도 추종 여부를 확인한다.

**Actual Speed 읽기 명령 — FC(04) Read Input Register**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x2D | 0x00 | 0x01 | CRC Lo | CRC Hi |

→ `01 04 D0 2D 00 01 99 CB`

### 판정 기준

- 아날로그 입력 전류(4~20mA)에 비례하여 팬 모터 속도/토크가 정상 제어됨
- 4mA: 최소 출력(0%), 20mA: 최대 출력(100%)에 해당
