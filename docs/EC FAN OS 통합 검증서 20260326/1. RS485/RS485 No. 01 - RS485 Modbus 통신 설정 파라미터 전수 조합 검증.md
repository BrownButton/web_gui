# RS485 No. 01 - RS485 Modbus 통신 설정 파라미터 (Baudrate / Parity / Stop Bit) 전수 조합 검증

## 시험 항목

EC FAN 드라이브가 지원하는 Baudrate × Parity × Stop Bit의 모든 조합에서 RS485 Modbus 통신이 정상 동작하는지 검증한다.

## 시험 목표

- 드라이브가 지원하는 모든 통신 설정 조합에서 Modbus RTU 통신이 성립됨을 확인한다.
- 설정 변경 후 전원 재투입 시 새 설정이 올바르게 적용됨을 확인한다.
- 시험 완료 후 초기 통신 설정(19200 / Even / Stop Bit 1)으로 완전히 복원됨을 확인한다.

## 시험 기준

### 사전 조건

| 항목 | 초기값 | 비고 |
|------|--------|------|
| Baudrate | **19200 bps** | 공장 출하 기본값 |
| Parity | **Even** | 공장 출하 기본값 |
| Stop Bit | **1** | Parity Even/Odd 시 1 |
| Node ID | 1 | |
| Motor Status | 정지, 알람 없음 | |

> 이전 시험에서 설정이 변경된 경우 반드시 초기화 후 진행.

### 지원 통신 설정 매트릭스

| 항목 | 지원 값 | 기본값 |
|------|---------|--------|
| Baudrate | 9600 / **19200** / 38400 / 57600 / 115200 | 19200 |
| Parity | **Even** / Odd / None | Even |
| Stop Bit | **1** / 2 | 1 |

### 통신 조합 규칙

| Parity | 권장 Stop Bit | 비고 |
|--------|--------------|------|
| Even | 1 | Modbus RTU 표준 |
| Odd | 1 | Modbus RTU 표준 |
| None | 2 | Modbus RTU 표준 |

### 통신 확인 방법

각 케이스에서 아래 파라미터 중 하나 이상을 FC03으로 읽어 정상 응답을 수신하면 합격으로 판정한다.

| 파라미터 | FC | 레지스터 주소 | 비고 |
|----------|----|-------------|------|
| Main Boot Ver | FC03 | 0xD002 | |
| Main FW Ver | FC03 | 0xD003 | |
| Inverter Boot Ver | FC03 | 0xD004 | |
| Inverter FW Ver | FC03 | 0xD005 | |

### 시험 절차

각 케이스에 대해 아래 공통 절차를 반복한다.

```
① 예지보전 툴 Setting 메뉴 → Baudrate / Parity / Stop Bit를 해당 케이스 값으로 설정
② Save to Memory 실행
③ 전원 Off → 3초 대기 → 전원 On (물리적 재투입)
④ 예지보전 툴에서 해당 케이스 설정으로 재접속
⑤ 통신 확인 파라미터 읽기 → 정상 응답 수신 여부 기록
⑥ 초기값(19200 / Even / Stop Bit 1)으로 복구 → Save → 전원 재투입 → 정상 접속 확인 후 다음 케이스 진행
```

#### [케이스 0] 초기값 기본 접속 확인 (Baseline)

- 설정: Baudrate 19200 / Parity Even / Stop Bit 1
- 예지보전 툴 접속 후 통신 확인 파라미터 읽기
- 이 케이스의 값을 이후 비교 기준으로 사용

#### [케이스 B] Baudrate 변경 (Parity=Even, Stop Bit=1 고정)

| 케이스 | Baudrate | Parity | Stop Bit |
|--------|----------|--------|----------|
| B-1 | 9600 | Even | 1 |
| B-2 | 38400 | Even | 1 |
| B-3 | 57600 | Even | 1 |
| B-4 | 115200 | Even | 1 |

#### [케이스 P] Parity / Stop Bit 변경 (Baudrate=19200 고정)

| 케이스 | Baudrate | Parity | Stop Bit |
|--------|----------|--------|----------|
| P-1 | 19200 | Odd | 1 |
| P-2 | 19200 | None | 2 |

#### [케이스 C] Baudrate + Parity 조합

| 케이스 | Baudrate | Parity | Stop Bit |
|--------|----------|--------|----------|
| C-1 | 9600 | Odd | 1 |
| C-2 | 9600 | None | 2 |
| C-3 | 38400 | Odd | 1 |
| C-4 | 38400 | None | 2 |
| C-5 | 57600 | Odd | 1 |
| C-6 | 57600 | None | 2 |
| C-7 | 115200 | Odd | 1 |
| C-8 | 115200 | None | 2 |

#### [최종] 초기값 복원 확인

모든 케이스 완료 후, 통신 설정이 초기값(19200 / Even / Stop Bit 1)인지 확인하고 예지보전 툴로 정상 접속 및 파라미터 응답을 확인한다.

### 판정 기준

| 케이스 | 판정 조건 |
|--------|----------|
| 케이스 0 | 초기값으로 정상 접속 및 파라미터 응답 수신 |
| 케이스 B-1 ~ B-4 | 각 Baudrate 변경 후 재접속 및 정상 응답 수신 |
| 케이스 P-1 ~ P-2 | 각 Parity/Stop Bit 변경 후 재접속 및 정상 응답 수신 |
| 케이스 C-1 ~ C-8 | 각 조합 변경 후 재접속 및 정상 응답 수신 |
| 최종 복원 | 초기값(19200 / Even / 1)으로 정상 복원 및 통신 확인 |

**합격 조건**: 지원하는 모든 케이스에서 정상 응답 수신 + 최종 복원 완료
