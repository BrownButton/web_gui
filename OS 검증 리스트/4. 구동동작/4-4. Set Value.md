# 4-4. Set Value

| 항목 | 내용 |
|------|------|
| **분류** | 구동동작 |
| **No** | 4-4 |

## 시험 목적

Set Point [0xD001] 파라미터를 통해 다양한 지령값을 설정하고 드라이브가 해당 값에 정상 추종하는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

### Open-loop 모드 Set Value 검증

1. Operating Mode를 Open-loop (0x0002)로 설정한다.
2. 다양한 Set Point 값을 설정하고 각 값에 대한 토크 출력을 확인한다.

| Set Point 값 | 의미 | 기대 토크 |
|-------------|------|-----------|
| 0 | 0% | 0% |
| 6554 (0x199A) | 10% | 10% |
| 32768 (0x8000) | 50% | 50% |
| 65535 (0xFFFF) | 100% | 100% |

- 변환 공식: raw = pct / 100 × 65535

3. Command Torque [0xD050] 파라미터를 읽어 설정값과 일치하는지 확인한다.

### Closed-loop Velocity 모드 Set Value 검증

4. Operating Mode를 Closed-loop Velocity (0x0000)로 설정한다.
5. 다양한 Set Point 값을 설정하고 각 값에 대한 속도 출력을 확인한다.

| Set Point 값 | 기대 속도(RPM) |
|-------------|--------------|
| 0 | 0 RPM |
| 3200 (0x0C80) | 100 RPM |
| 6400 (0x1900) | 200 RPM |
| 32000 (0x7D00) | 1000 RPM |

- 변환 공식: raw = rpm / maxSpeed × 64000 (maxSpeed = 1600 RPM 기준)

6. Actual Speed [0xD02D]를 읽어 설정 속도에 추종하는지 확인한다.

## 판정기준

- Set Point 값에 따라 토크/속도가 정상 비례하여 제어됨
- Command Torque / Actual Speed 파라미터가 기대값에 일치 또는 추종

## 시험결과

| Set Point | 모드 | 기대값 | 실측값 | 판정 |
|-----------|------|--------|--------|------|
| 0 | Open-loop | 0% | - | 합격 / 불합격 |
| 6554 | Open-loop | 10% | - | 합격 / 불합격 |
| 3200 | Velocity | 100 RPM | - | 합격 / 불합격 |
