# 3-8. 펌웨어 버전 확인 (Main / Main Boot)

| 항목 | 내용 |
|------|------|
| **분류** | 기본동작 |
| **No** | 3-8 |

## 시험 목적

Main 펌웨어 버전 및 Main Boot 버전 파라미터가 정상적으로 출력되는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 또는 EC FAN Control 프로그램

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. EC FAN Control 프로그램 또는 Hercules를 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. COM Port 연결을 확인한다.
4. Main SW Ver 파라미터를 읽는다.

**Main SW Version 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | Main Ver Addr Hi | Main Ver Addr Lo | 0x00 | 0x01 | CRC Hi | CRC Lo |

5. Main Boot Ver 파라미터를 읽는다.

**Main Boot Version 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | Main Boot Ver Addr Hi | Main Boot Ver Addr Lo | 0x00 | 0x01 | CRC Hi | CRC Lo |

6. 읽은 버전이 현재 탑재된 펌웨어 버전과 일치하는지 확인한다.
7. OS 다운로드 후 버전이 갱신되는지 확인한다. (3-10-1 시험과 연계)

## 판정기준

- Main SW Ver 파라미터가 현재 탑재된 펌웨어 버전과 일치
- Main Boot Ver 파라미터가 정상 출력

## 시험결과

| 항목 | 기대값 | 실측값 | 판정 |
|------|--------|--------|------|
| Main SW Ver | - | - | 합격 / 불합격 |
| Main Boot Ver | - | - | 합격 / 불합격 |

## 기타

Main / Main Boot 버전 파라미터 데이터 캡처 첨부
