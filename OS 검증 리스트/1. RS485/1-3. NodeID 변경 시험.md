# 1-3. NodeID 변경 시험

| 항목 | 내용 |
|------|------|
| **분류** | RS485 |
| **No** | 1-3 |

## 시험 목적

예지보전 툴을 이용해 Node ID 설정이 되는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
- USB to RS485 Converter
- EC FAN Configurator (EC FAN Control 프로그램)

## 시험 방법

1. USB to RS485 컨버터를 이용하여 EC FAN과 노트북을 서로 연결한다.
2. EC FAN Control 프로그램을 실행 후 COM Port: USB to RS485 Converter 접속 포트와 Baudrate: 19200, Parity: Even으로 설정 후 접속 버튼을 누른다.
   - (Baudrate 19,200bps, Even Parity 조건은 공장 출하 상태에서의 초기 값이며, 이 값이 팬 드라이브 설정과 다를 경우 접속이 되지 않으므로 유의)
3. Setting 메뉴에 진입 후 Node ID 항목에 0을 입력한다.
   - (설정된 Node의 파라미터가 갱신되는 구조, Broadcasting ID를 통해 설정된 Node ID 확인 목적)
4. [0xD100] Fan Address의 값을 1로 설정한다.
5. Save to Memory 버튼을 눌러 파라미터를 저장한다.
6. 전원 재 투입 후 통신을 재 연결 한다.
7. Setting 메뉴의 Node ID에 1을 입력한다.
8. 파라미터가 올라오는 것을 확인한다.

## 판정기준

- 설정 Node ID 입력 시 Device, SW Ver, Boot Ver 출력

## 시험결과

- [ ] 합격 / 불합격
