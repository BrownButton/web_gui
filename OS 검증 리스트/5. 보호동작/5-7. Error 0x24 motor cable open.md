# 5-7. Error 0x24 Motor Cable Open

| 항목 | 내용 |
|------|------|
| **분류** | 보호동작 |
| **No** | 5-7 |

## 시험 목적

모터 케이블이 단선(Open) 되었을 때 드라이브가 보호 동작(알람)을 수행하는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)

## 시험 방법

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. 모터와 드라이브 사이의 UVW 케이블 중 하나를 분리한다.
   - (안전을 위해 전원 Off 상태에서 케이블 분리 후 전원 On)
4. 전원을 인가하고 구동 명령을 전송한다.
5. Motor Status [0xD011]을 읽어 에러 비트 Set 여부를 확인한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

6. Error 코드 파라미터를 읽어 0x24 (Motor Cable Open)가 발생했는지 확인한다.
7. 케이블 재연결 후 Alarm Reset을 실행하여 정상 복귀 확인한다.

## 판정기준

- 모터 케이블 단선 시 드라이브가 보호 동작을 수행함
- Error 코드 0x24 (Motor Cable Open) 확인

## 시험결과

- [ ] 합격 / 불합격

## 기타

- 시험 후 모터 케이블을 반드시 재연결하고 정상 동작 확인
- 케이블 단선 시 Motor Status 데이터 캡처 첨부
