# 5-8. Error 0x25 Drive Temperature 2

| 항목 | 내용 |
|------|------|
| **분류** | 보호동작 |
| **No** | 5-8 |

## 시험 목적

드라이브(인버터부) 온도가 과온 임계값을 초과했을 때 보호 동작(알람)이 수행되는지 확인한다.

## 적용 모델

- EC-FAN ✓

## 시험 장비

- EC FAN 1EA
  - (Node Address를 1로 설정한 상태에서 시험한다. 설정 방법은 RS485 No. 1-3 시험 참조)
- USB to RS485 Converter
- Hercules v3.2.8 (테스트 프로그램)
- 열풍기 또는 환경 챔버

## 시험 방법

> 📌 **참고**: Error 0x22와 Error 0x25 중 인버터 관련 항목은 삭제하고 하나를 유지하는 방향으로 검토 중

1. USB to RS485 Converter를 이용하여 EC FAN과 노트북을 연결한다.
2. Hercules v3.2.8 테스트 프로그램을 실행 후 통신 설정을 진행한다.
   - Baud: 19200bps, Data Size: 8, Parity: even
3. 드라이브를 정격 부하로 구동하거나 열풍기로 인버터부 온도를 상승시킨다.
4. IGBT 온도 파라미터를 모니터링하면서 과온 임계값에 근접하는지 확인한다.
5. Motor Status [0xD011]을 읽어 TFE(인버터 과열) 비트 Set 여부를 확인한다.

**Motor Status 읽기 명령**

| 바이트 | 0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 |
|--------|------|------|------|------|------|------|------|------|
| 내용 | 0x01 | 0x04 | 0xD0 | 0x11 | 0x00 | 0x01 | 0x59 | 0x0F |

6. Error 코드 파라미터를 읽어 0x25 (Drive Temperature 2)가 발생했는지 확인한다.
7. 냉각 후 Alarm Reset을 실행하여 정상 복귀 확인한다.

## 판정기준

- 인버터부 과온 조건 발생 시 보호 동작을 수행함
- Error 코드 0x25 (Drive Temperature 2) 또는 Motor Status TFE 비트 Set 확인

## 시험결과

- [ ] 합격 / 불합격

## 기타

- IGBT 온도 파라미터 추이 및 에러 발생 시 Motor Status 데이터 첨부
